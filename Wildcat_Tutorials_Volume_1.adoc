= About this Book

This book is a collaborative effort by students at Northern Michigan University. The tutorials are licensed using the Creative Commons Attribution Share Alike 4.0. These tutorials can be freely distributed, but please attribute the authors.

Some of the topics include tutorials regarding network security. The tutorials are presented for informational and instructional purposes only. These tutorials should only be conducted on systems for which you have authorization.


= Wildcat Tutorials

The remainder of this book is divided into several chapters. Each chapter is a standalone tutorial that focuses on one tool or technique.
